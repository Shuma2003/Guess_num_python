# Документация: Игра "Угадай число" (версия с HP)
Обзор

Консольная игра "Угадай число" с уникальной механикой здоровья (HP). Игрок должен угадать случайное число, теряя HP при каждой неудачной попытке. Если HP достигает нуля - игра заканчивается поражением.
Особенности

    Механика здоровья (HP): Начальный HP = 100, при каждой ошибке теряется 10 HP

    Ограниченное количество попыток: Максимум 10 попыток (100 HP / 10 за попытку)

    Автоматическое раскрытие числа: При поражении показывается загаданное число

    Простой интерфейс: Минималистичный дизайн без сложных меню

Установка и запуск
bash

# Скачайте файл game.py
# Запустите программу:
python game.py

Как играть

    Программа загадывает случайное число от 1 до 5

    У вас есть 100 HP (10 попыток)

    Вводите свои предположения

    За каждую ошибку теряете 10 HP

    Цель: угадать число до того, как HP достигнет 0

Код программы с комментариями
python

import random

# Программа загадывает число от 1 до 5
print("Программа загадала число от 1 до 10")  # Примечание: в сообщении диапазон 1-10, но в коде 1-5
secret = random.randint(1, 5)  # Генерация случайного числа

# Инициализация здоровья игрока
hp = 100

# Основной игровой цикл
while hp > 0:
    # Получение предположения от игрока
    guess = int(input("Попробуйте угадать число"))
    
    # Проверка угадал ли игрок
    if guess == secret:
        print("Число отгадано")
        break  # Выход из цикла при победе
    else:
        # Игрок не угадал - теряет HP
        hp = hp - 10
        print(f"Не отгадал, Осталось HP: {hp} ")
        
        # Проверка на поражение
        if hp <= 0:
            print(f"Загаданное число {secret}")  # Раскрытие секретного числа

Особенности реализации
Ограничения

    Диапазон чисел: В коде random.randint(1, 5), но в сообщении указано "от 1 до 10"

    Штраф за ошибку: Фиксированный -10 HP

    Максимальное число попыток: 10 (100 HP / 10)

Потенциальные проблемы

    Нет обработки ошибок ввода: Если ввести не число, программа завершится с ошибкой

    Нет проверки диапазона: Можно ввести число вне диапазона 1-5

    Противоречие в сообщении: Указано "от 1 до 10", но реальный диапазон 1-5

Версия 1.1 - с обработкой ошибок
python

import random

print("Программа загадала число от 1 до 5")
secret = random.randint(1, 5)
hp = 100

while hp > 0:
    try:
        guess = int(input("Попробуйте угадать число (1-5): "))
        
        if guess < 1 or guess > 5:
            print("Число должно быть от 1 до 5!")
            continue
            
        if guess == secret:
            print("Число отгадано")
            break
        else:
            hp = hp - 10
            print(f"Не отгадал, Осталось HP: {hp}")
            
            if hp <= 0:
                print(f"Загаданное число {secret}")
                
    except ValueError:
        print("Пожалуйста, введите число!")
